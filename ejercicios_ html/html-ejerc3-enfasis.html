Trabajo sobre la web 2.0 y 3.0

Abstract

La asignatura TIC II propone el desarrollo de un trabajo de investigación sobre la evolución de la

web para comprender el contexto social y tecnológico del nacimiento de Internet. Esto es necesario

debido a que el currículo de 2º de bachillerato incluye el desarrollo de páginas webs.


En este trabajo se recogen las generalidades sobre las distintas versiones de la web

como su origen, características o ventajas. Se desarrolla la evolución de las páginas webs como la web 1.0

que se caracteriza por ser la primera en aportar información al usuario después hace un desarrollo a

la web 2.0 donde el usurario se hace más presente a la hora de interactuar con las páginas hasta

llegar a la actual web 3.0 o web semántica que personaliza las búsquedas de los usuarios utilizando

inteligencia artificial. Para recoger estos aspectos se ha hecho uso de fuentes de información

disponibles en diferentes webs.
